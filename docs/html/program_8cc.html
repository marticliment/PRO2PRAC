int main ( )

The entry point of the program.

This function initializes the Valley from the input stream and processes commands until the "fin" command is entered. It reads commands from the input stream and performs the corresponding operations based on the command.

Precondició
cin is readable and cout is writable
Postcondició
The program will read and execute commands to simulate commerce in a valley
Retorna
The exit status of the program.
